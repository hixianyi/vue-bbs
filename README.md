# vue-bbs
本项目是：Vue框架的移动端的论坛（普通用户权限/管理员权限）  

## 项目背景  
本项目是个人练习的作品。所有代码均为自己编写。  
界面模仿、改进自“百度贴吧-极速版APP”。  

## 技术栈  
【前端】Vue+Vue-Router+Vuex+WebPack+Mint UI+Axios  
【后台】PHP
