# vue-bbs
本项目是：Vue框架的移动端的论坛（前台+后台）。 

## 项目背景  
本项目是个人练习的作品。前台/后台代码均为自己编写。  
前台界面模仿“百度贴吧-极速版APP”。  
后台界面为自制。  

## 技术栈  
【前端】Vue+Vue-Router+Vuex+WebPack+Mint UI+Axios  
【后台】PHP
